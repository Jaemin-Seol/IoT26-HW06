# IoT26-HW06 Team C (Cho Wooyoung, Seol Jaemin, Lim Youbin)
We choose to implement Node-RED and YOLO to detect cars.   

Due to version difference in Node-RED, minor changes were made.   
The structure of project is simple.   
1. Node-RED Dashboard (template node) sends button click.
2. Python script is executed and save image at predefind directory.
3. Dashboard shows the image.
